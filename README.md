<h2>Technology Stack</h2>

<p>
AgriSmart AI is built using a modern, scalable, and AI-ready technology stack. The frontend is developed using <strong>Next.js</strong> with <strong>TypeScript</strong>, <strong>Tailwind CSS</strong>, and <strong>ShadCN UI</strong> to create a fast, responsive, and user-friendly experience. The backend is powered by <strong>Python</strong> using <strong>FastAPI</strong> and REST APIs for efficient and high-performance server-side operations.
</p>

<p>
The platform uses <strong>PostgreSQL via Supabase</strong> as its primary database, along with Supabase Authentication and Storage for secure user management and file handling. Artificial Intelligence and Machine Learning functionalities are implemented using <strong>TensorFlow</strong>, <strong>PyTorch</strong>, <strong>Scikit-Learn</strong>, and <strong>OpenCV</strong> for crop disease detection, crop quality assessment, and intelligent recommendation systems.
</p>

<p>
For voice-based interactions and multilingual accessibility, the system integrates <strong>Whisper Speech-to-Text</strong>, <strong>gTTS Text-to-Speech</strong>, and <strong>IndicTrans2</strong> for regional language translation and voice responses. Real-time weather insights and weather-based farming alerts are powered through the <strong>OpenWeatherMap API</strong>.
</p>

<p>
Version control and collaboration are managed using <strong>Git</strong> and <strong>GitHub</strong>, while deployment and hosting are handled through <strong>Vercel</strong> and <strong>Render</strong>, ensuring scalability, reliability, and seamless continuous deployment.
</p>
